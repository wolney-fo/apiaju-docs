## What is ApiAju?

ApiAju is a free, public REST API designed to provide comprehensive calendar information for Aracaju, the capital city of Sergipe state in Brazil. Access accurate data about holidays, business days, and date calculations specific to the Aracaju region.

## Quick Start

Get started with ApiAju in seconds. No registration or API keys required!

<CodeGroup>

```bash cURL
curl https://api.aju.br/api/v1/calendar/holidays?year=2024
```

```javascript JavaScript
const response = await fetch('https://api.aju.br/api/v1/calendar/holidays?year=2024');
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.aju.br/api/v1/calendar/holidays?year=2024')
data = response.json()
print(data)
```

</CodeGroup>

## Main Features

<AccordionGroup>
  <Accordion title="ðŸŽ‰ Holidays Information" icon="calendar-star">
    Get complete lists of holidays for any year between 1900-2199, including:
    - **National holidays**: Valid throughout Brazil
    - **State holidays**: Specific to Sergipe
    - **Municipal holidays**: Exclusive to Aracaju

    Perfect for scheduling systems, calendar applications, and business planning tools.
  </Accordion>

  <Accordion title="ðŸ“Š Business Days Calculator" icon="business-time">
    Calculate business days (working days) between two dates, automatically excluding:
    - Weekends (Saturdays and Sundays by default)
    - All registered holidays
    - Optional Saturday inclusion for businesses that work on Saturdays

    Essential for project management, SLA calculations, and deadline tracking.
  </Accordion>

  <Accordion title="ðŸŽ¯ Holiday Verification" icon="circle-check">
    Quickly check if any specific date is a holiday in Aracaju. Returns detailed information including:
    - Whether the date is a holiday
    - The holiday name and type
    - The holiday category (national/state/municipal)
  </Accordion>

  <Accordion title="ðŸ“… End Date Calculation" icon="forward">
    Calculate the end date by adding a specific number of business days to a start date. Perfect for:
    - Project deadline calculations
    - Delivery date estimations
    - Service level agreement (SLA) tracking
  </Accordion>
</AccordionGroup>
