Welcome! This guide will help you make your first API call and understand the core features of ApiAju.

## Base URL

All API requests are made to:

```
https://api.aju.br/api/v1
```

<Note>
  **No Authentication Required**: ApiAju is a public API. You can start making requests immediately without any API keys or registration.
</Note>

## Your First API Call

Let's fetch the holidays for the current year:

<CodeGroup>

```bash cURL
curl https://api.aju.br/api/v1/calendar/holidays
```

```javascript JavaScript (Fetch)
const response = await fetch('https://api.aju.br/api/v1/calendar/holidays');
const data = await response.json();
console.log(data);
```

```javascript JavaScript (Axios)
const axios = require('axios');

const response = await axios.get('https://api.aju.br/api/v1/calendar/holidays');
console.log(response.data);
```

```python Python
import requests

response = requests.get('https://api.aju.br/api/v1/calendar/holidays')
data = response.json()
print(data)
```

```php PHP
<?php
$response = file_get_contents('https://api.aju.br/api/v1/calendar/holidays');
$data = json_decode($response, true);
print_r($data);
?>
```

```java Java
import java.net.http.*;
import java.net.URI;

HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.aju.br/api/v1/calendar/holidays"))
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

</CodeGroup>

## Core Features

Now let's explore the main features of ApiAju:

<Steps>
  <Step title="Get Holidays for a Specific Year">
    Retrieve all holidays for any year between 1900-2199:

    ```bash
    curl "https://api.aju.br/api/v1/calendar/holidays?year=2024"
    ```

    <Accordion title="View Response">
      ```json
      {
        "message": "Listing the holidays of 2024",
        "data": [
          {
            "date": "2024-01-01",
            "name": "Confraternização mundial",
            "type": "national"
          },
          {
            "date": "2024-12-25",
            "name": "Natal",
            "type": "national"
          }
          // ... all holidays for 2024
        ]
      }
      ```
    </Accordion>
  </Step>

  <Step title="Check if a Date is a Holiday">
    Verify if a specific date is a holiday:

    ```bash
    curl "https://api.aju.br/api/v1/calendar/holidays/is-holiday?date=2024-12-25"
    ```

    <Accordion title="View Response">
      ```json
      {
        "data": {
          "date": "2024-12-25",
          "isHoliday": true
        }
      }
      ```
    </Accordion>

    <Tip>
      If you don't provide a date parameter, it checks the current date.
    </Tip>
  </Step>

  <Step title="Calculate Business Days Between Dates">
    Calculate the number of working days between two dates:

    ```bash
    curl "https://api.aju.br/api/v1/calendar/business-days?startDate=2024-01-01&endDate=2024-12-31"
    ```

    <Accordion title="View Response">
      ```json
      {
        "data": {
          "businessDays": 252
        }
      }
      ```
    </Accordion>

    <Info>
      By default, Saturdays are excluded. Use `includeSaturdays=true` to include them as working days.
    </Info>
  </Step>

  <Step title="Calculate End Date from Business Days">
    Find the end date by adding a number of business days to a start date:

    ```bash
    curl "https://api.aju.br/api/v1/calendar/business-days/end-date?startDate=2024-01-01&workingDays=30"
    ```

    <Accordion title="View Response">
      ```json
      {
        "data": {
          "endDate": "2024-02-13"
        }
      }
      ```
    </Accordion>

    <Tip>
      Perfect for calculating project deadlines or delivery dates!
    </Tip>
  </Step>
</Steps>
