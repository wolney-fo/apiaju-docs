## Overview

ApiAju provides a RESTful API for accessing calendar, holiday, and business day information for Aracaju-SE, Brazil. All endpoints are publicly accessible with no authentication required.

## API Endpoints

ApiAju provides the following endpoint categories:

### Health Check

Monitor the API status and availability.

- **GET** `/health` - Check if the API is running

### Calendar & Holidays

Access holiday information for Aracaju.

- **GET** `/calendar/holidays` - List all holidays for a year
- **GET** `/calendar/holidays/is-holiday` - Check if a date is a holiday

### Business Days

Calculate working days and date ranges.

- **GET** `/calendar/business-days` - Calculate business days between dates
- **GET** `/calendar/business-days/end-date` - Calculate end date from business days

<Note>
  All calendar endpoints automatically handle national (Brazil), state (Sergipe), and municipal (Aracaju) holidays.
</Note>

## Date Formats

All dates in the API use the ISO 8601 date format:

```
YYYY-MM-DD
```

**Examples:**
- `2024-01-01` - January 1, 2024
- `2024-12-25` - December 25, 2024

## Response Structure

### Success Response

All successful responses follow this structure:

```json
{
  "message": "Descriptive message about the operation",
  "data": {
    // Response data here
  }
}
```

### Error Response

Error responses include helpful error messages:

```json
{
  "message": "High-level error description",
  "error": "Detailed error information"
}
```

<Warning>
  Always check the HTTP status code to determine if a request was successful (200-299 range).
</Warning>

## Holiday Types

Holidays in ApiAju are categorized by their scope:

<AccordionGroup>
  <Accordion title="National (national)" icon="flag">
    Holidays celebrated throughout Brazil.
    
    **Examples:**
    - New Year's Day (Confraternização mundial)
    - Independence Day (Independência do Brasil)
    - Christmas (Natal)
  </Accordion>

  <Accordion title="State (state)" icon="map">
    Holidays specific to Sergipe state.
    
    **Examples:**
    - State-specific commemorations
  </Accordion>

  <Accordion title="Municipal (municipal)" icon="building">
    Holidays specific to Aracaju city.
    
    **Examples:**
    - City Anniversary (Aniversário da cidade) - March 17
  </Accordion>
</AccordionGroup>

## Year Range

The API supports calculations for years between **1900** and **2199**.

<Info>
  Requests outside this range will return a `400 Bad Request` error with details about the valid range.
</Info>
